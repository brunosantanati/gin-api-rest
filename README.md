### Comandos

```
go mod init github.com/brunosantanati/api-go-gin
go get -u github.com/gin-gonic/gin
docker-compose up
docker-compose down

Executar:
docker-compose exec postgres sh
E depois executar esse outro comando para pegar o ip:
hostname -i
```
